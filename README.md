# 鸿蒙课表管理系统 java230

本项目是一个基于鸿蒙操作系统（HarmonyOS）的课表管理系统，采用ArkTS和后端Spring Boot技术栈进行开发。

## 技术栈

- **前端**：使用鸿蒙的ArkTS和ArkUI进行界面设计和功能实现。
- **后端**：采用Spring Boot框架进行业务逻辑处理和数据存储。

## 功能模块

- 课表展示
- 课程管理
- 时间提醒
- 用户权限控制

## 系统角色

- 学生：查看课表、接收时间提醒。
- 教师：管理课程、发布通知。
- 管理员：管理用户、课程及系统设置。

## 运行环境

- **开发工具**：华为鸿蒙的DevEco Studio。
- **适用平台**：鸿蒙操作系统。

## 部署步骤

1. 环境搭建：安装DevEco Studio及相关插件。
2. 代码编译：使用DevEco Studio进行编译。
3. 应用部署：将编译后的应用部署到鸿蒙设备。
